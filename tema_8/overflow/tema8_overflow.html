Ángel Baena del Río · Overflow

Introducción

Controla el comportamiento de un contenedor cuando su contenido es más grande que el contenedor mismo y se desborda.

Valores de Overflow

visible

El valor visible es el valor por defecto. El contenido que se desborda simplemente
se muestra fuera del contenedor, sobresaliendo visualmente.

hidden

El valor hidden es el contenido que se desborda está oculto y no es visible. Esto puede ser útil cuando no quieres
que el contenido que sobresale sea visible.

scroll

El valor scroll hace que se agregue una barra de desplazamiento al contenedor,
permitiendo al usuario desplazarse para ver el contenido que está fuera de la vista.

auto

El valor auto es similar a scroll, pero se agregarán barras de desplazamiento solo cuando sea necesario.
Si el contenido se ajusta dentro del contenedor, no se mostrarán barras de desplazamiento.
